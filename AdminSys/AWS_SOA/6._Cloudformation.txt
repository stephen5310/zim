====== 6. Cloudformation ======
Created mardi 07 janvier 2025

* AWS official IaC tool that tags automatically ressources for costs evaluation (Regional)
* Templates must be uploaded in S3 and then referenced in CloudFormation
	* Can use the console or awscli
* Create a new version of a template to update a stack
* Stacks are identified by a unique name within a region
