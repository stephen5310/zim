====== AWS SDK, IAM Roles and Policies ======

* To create IAM policies :
	* Use JSON
	* Use the visual editor
	* Use AWS Policy Generator
* To test IAM policies
	* Use IAM Policy Simulator
	* Use the CLI :
		* To retrieve EC2 Instance Metadata from http://169.254.169.254/latest/meta-data/
		* Cannot retrieve IAM Policy from the previous url
* To attach policies to IAM roles
* SDK API calls use us-east-I region by default if you don't specify
